Ініціалізація репозиторію: git init.
Перевірка статусу репозизорію: git status
Прив*язка віддаленого репозиторію: git remote add origin https:..
Показує список підключених з*єднань: git remote -v
додавання файлів в індекс (для трекання): git add назва папки/файла
записати коміт з повідомленням: git commit -m 'повідомлення'
створити ssh ключ: ssh-keygen
відправити на віддалений сервер: git push
